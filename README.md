# Dynamic Casino

Приложение казино на базе JavaFX с модульной архитектурой и динамической механикой игр.

[![Java](https://img.shields.io/badge/Java-17-orange.svg)](https://www.oracle.com/java/)
[![Maven](https://img.shields.io/badge/Maven-3.8+-blue.svg)](https://maven.apache.org/)

## Описание

Dynamic Casino — это настольное приложение казино, разработанное на Java и JavaFX. Оно предоставляет современный графический интерфейс, расширяемую архитектуру игр и профессиональные алгоритмы расчета вероятностей для реалистичного игрового опыта.

## Возможности

- **Графический пользовательский интерфейс**: Современный интерфейс на основе FXML с адаптивным дизайном
- **Модульная архитектура**: Система на основе плагинов для добавления новых игр
- **Динамическое управление играми**: Гибкая обработка состояния игры и переходы
- **Алгоритмы вероятностей**: Настраиваемые расчеты вероятностей
- **Генерация случайных чисел**: Настраиваемые генераторы случайных чисел
- **Паттерн Factory для символов**: Простое управление символами и расширяемость

## Требования

- Java 11 и выше
- Maven 3.6 и выше
- JavaFX SDK

## Архитектура

### Основные компоненты

**Main.java**
- Точка входа приложения
- Инициализирует жизненный цикл приложения JavaFX

**CasinoController.java**
- Обрабатывает взаимодействие с пользовательским интерфейсом
- Управляет обновлениями интерфейса и обработкой событий

**GameState.java**
- Поддерживает текущее состояние игры
- Управляет переходами игры и сохранением данных

**Command.java**
- Реализует паттерн проектирования Command
- Позволяет гибко выполнять действия и функции отмены/повтора

**SymbolFactory.java**
- Создает символы игры с использованием паттерна Factory
- Поддерживает расширяемые типы символов

**ProbabilityCalculator и DefaultProbabilityCalculator**
- Расчеты вероятностей на основе интерфейса
- Позволяет пользовательские реализации вероятностей

**RandomNumberGenerator и DefaultRandomGenerator**
- Подключаемое генерирование случайных чисел
- Поддерживает генерирование с зaданным семенем для тестирования

## Разработка

### Добавление новых игр

1. Реализуйте интерфейс `Command` для логики новой игры
2. Используйте `SymbolFactory` для создания символов игры
3. Добавьте компоненты пользовательского интерфейса в `casino.fxml`
4. Зарегистрируйте новую игру в `CasinoController`

### Добавление пользовательского расчета вероятностей

Расширьте интерфейс `ProbabilityCalculator` для реализации пользовательских алгоритмов вероятностей.

### Пользовательское генерирование случайных чисел

Реализуйте интерфейс `RandomNumberGenerator` для альтернативных стратегий генерирования случайных чисел.
